Baixar corpus de Janeiro de Yellow Taxi Trip Records (CSV)
https://www1.nyc.gov/site/tlc/about/tlc-trip-record-data.page

Executar o comando python3 taxi_corridas_dia.py nome_arquivo (Exemplo: yellow_tripdata_2019-01.csv)
Executar o comando python3 taxi_media_minutos.py nome_arquivo (Exemplo: yellow_tripdata_2019-01.csv)
Executar o comando python3 taxi_media_valor.py nome_arquivo (Exemplo: yellow_tripdata_2019-01.csv)
